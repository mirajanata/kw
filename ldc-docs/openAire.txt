-> OpenAIRE Graph (Linked Open Data, api.opencitations.net/meta/v1)
- query organisations (list of IDs in config.js), their projects plus related organisations
- extract info of all organisations (oa_ID, shortName, name, homepage, country) and all projects (oa_ID, name, description, acronym (ascii), startDate, endDate, budget), and relations
- calculate new URIs for organisations (host domain) and projects (acronym)
- create keywords for projects (name, description using Keyword Assistant js)
- transform into NT triples structure
